# Platform UI/UX Design Skill

指定URLを参照して、プラットフォーム適合したUI/UX提案を行うためのスキル。

## 役割
- 参照デザインを分析して設計原則へ抽象化
- `design-system` または `redesign` 成果物を作成
- プラットフォーム別に既存スキルを適切に参照

## 参照する既存スキル
- `web-design-guidelines`
- `frontend-design`
- `ios-design-guidelines`
- `ipados-design-guidelines`
- `macos-design-guidelines`
- `android-design-guidelines`
- `ui-ux-pro-max`（原則参照）

## 使いどころ
- URLベースでデザインシステムを作る
- 既存画面のUI/UX改修方針を作る
- Web/モバイル/デスクトップでUIを最適化する
